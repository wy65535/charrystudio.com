隐私政策

我们重视并保护您的隐私

最后更新日期：2026年1月11日

欢迎访问 CharryStudio.com（以下简称"本站"或"我们"）。我们深知隐私对您的重要性，并致力于保护您的个人信息和隐私安全。本隐私政策说明了我们如何收集、使用、存储和保护您的信息。

重要提示：本站是 Cherry Studio 的独立推广网站，不是 Cherry Studio 官方网站。本隐私政策仅适用于 CharryStudio.com 网站。关于 Cherry Studio 软件本身的隐私政策，请访问官方网站 cherry-ai.com。

1. 信息收集

1.1 我们收集的信息

本站是一个静态信息展示网站，我们收集的信息非常有限：

  • 访问日志：当您访问本站时，我们的服务器会自动记录标准的访问日志信息，包括 IP 地址、浏览器类型、访问时间、访问的页面等。
  • Cookie：我们可能使用 Cookie 来改善用户体验和分析网站流量。
  • 分析数据：我们可能使用第三方分析工具（如 Google Analytics）来了解网站使用情况。

1.2 我们不收集的信息

本站不会主动收集以下信息：

  • 个人身份信息（姓名、地址、电话号码等）
  • 支付信息
  • 账号密码
  • 其他敏感个人信息

2. 信息使用

我们收集的信息用于以下目的：

  • 改善网站：了解用户如何使用本站，以便改进内容和功能
  • 分析流量：分析访问模式和趋势，优化用户体验
  • 技术维护：诊断技术问题，维护网站安全
  • 法律合规：遵守适用的法律法规要求

3. Affiliate 联盟声明

重要披露：本站包含联盟链接（Affiliate Links）。当您通过本站的链接注册或购买服务时，我们可能会获得佣金。

3.1 联盟关系

CharryStudio.com 是 Cherry Studio 及相关服务的推广合作伙伴。我们在网站中提供的某些链接包含联盟跟踪代码，用于识别来自本站的访问和转化。

3.2 对您的影响

  • 通过联盟链接注册或购买不会增加您的成本
  • 联盟佣金由服务提供商支付，不会从您的账户扣除
  • 您享受的服务和价格与直接访问官网完全相同
  • 联盟链接中的跟踪参数仅用于识别来源，不会收集您的个人信息

3.3 推荐原则

我们承诺：

  • 仅推荐我们真正使用、测试和相信的产品和服务
  • 提供真实、客观的产品信息和评价
  • 不夸大产品功能或效果
  • 以用户最佳利益为出发点提供推荐

4. Cookie 和跟踪技术

4.1 Cookie 使用

Cookie 是存储在您设备上的小型文本文件。我们使用 Cookie 用于：

  • 记住您的偏好设置
  • 分析网站使用情况
  • 跟踪联盟链接的转化

4.2 管理 Cookie

您可以通过浏览器设置管理或删除 Cookie。请注意，禁用 Cookie 可能影响网站的某些功能。大多数浏览器允许您：

  • 查看已存储的 Cookie
  • 删除所有或特定 Cookie
  • 阻止第三方 Cookie
  • 设置 Cookie 过期时间

5. 第三方服务

5.1 外部链接

本站包含指向第三方网站的链接，包括但不限于：

  • Cherry Studio 官方网站 (cherry-ai.com)
  • GitHub 仓库
  • API 服务提供商
  • 社交媒体平台

这些第三方网站有自己的隐私政策，我们不对其隐私做法负责。我们建议您在访问这些网站时查看它们的隐私政策。

5.2 分析工具

我们可能使用以下第三方分析工具：

  • Google Analytics：用于网站流量分析
  • Cloudflare Analytics：用于网站性能监控

这些工具可能会收集和处理有关您访问的信息。它们的数据处理受各自隐私政策约束。

6. 数据安全

我们采取合理的技术和组织措施来保护收集的信息：

  • 使用 HTTPS 加密传输数据
  • 通过 Cloudflare CDN 提供额外安全保护
  • 定期更新安全措施
  • 限制对数据的访问权限

然而，请注意互联网传输永远无法保证 100% 安全。我们会尽最大努力保护您的信息，但无法保证绝对安全。

7. 数据保留

我们仅在必要期间保留收集的信息：

  • 访问日志：通常保留 30-90 天
  • 分析数据：根据分析工具的政策，通常为 14-26 个月
  • Cookie：根据 Cookie 类型，从会话结束到最多 2 年

8. 您的权利

根据适用的数据保护法律，您可能拥有以下权利：

  • 访问权：了解我们收集了哪些关于您的信息
  • 更正权：要求更正不准确的信息
  • 删除权：要求删除您的个人信息
  • 反对权：反对某些数据处理活动
  • 撤回同意：随时撤回您的同意

由于本站收集的信息非常有限且大多为匿名数据，我们可能无法识别特定用户的数据。如有疑问，请联系我们。

9. 儿童隐私

本站不针对 13 岁以下儿童，我们不会故意收集 13 岁以下儿童的个人信息。如果您认为我们可能收集了儿童的信息，请立即联系我们。

10. 国际数据传输

本站通过 Cloudflare CDN 提供服务，您的信息可能会被传输到您所在国家/地区以外的服务器。这些地区的数据保护法律可能与您所在地区不同。通过使用本站，您同意这种数据传输。

11. Cherry Studio 软件隐私

关于 Cherry Studio 软件：

Cherry Studio 桌面客户端软件采用本地优先的隐私保护策略：

  • 所有对话数据和设置存储在您的本地设备上
  • 不会将您的对话内容上传到 Cherry Studio 的服务器
  • 当您使用 AI 模型时，数据直接发送给您选择的 AI 服务提供商
  • 您对自己的数据拥有完全控制权

详细的软件隐私政策请访问 Cherry Studio 官方网站。

我们可能会不时更新本隐私政策。重大变更时，我们会在网站上发布通知。建议您定期查看本页面以了解最新信息。继续使用本站即表示您接受更新后的隐私政策。

本隐私政策受适用法律管辖。我们努力遵守以下法规：

  • 欧盟通用数据保护条例（GDPR）
  • 加州消费者隐私法案（CCPA）
  • 中国网络安全法及相关法规
  • 其他适用的数据保护法律

使用 CharryStudio.com 网站即表示您已阅读、理解并同意本隐私政策。如果您不同意本政策的任何部分，请停止使用本站。

我们承诺保持透明的数据处理实践。本站的主要目的是提供 Cherry Studio 的信息和推广服务，帮助用户了解和使用这款优秀的 AI 工具。我们重视您的信任，并将持续保护您的隐私。
